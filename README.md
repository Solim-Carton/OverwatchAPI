## OverwatchAPI

A RESTful API built to help overwatch players understand the matchups that take place in the game

## Setup Instructions

2. Install dependencies: `npm install`
4. Setup the database: `npm run setup`
5. Seed the database: `npm run seed`
6. Start the server: `npm start`

## Register Users

1. Registar user with username and password
2. login with user
3. use API

## Guide 
1. /api/auth/register -Creates user 
2. /api/auth/login -Login user
3. /api/heroes -Returns all Heroes
4. /api/heroes/:id -Get heroes by id
5. /api/heroes/search -Search heroes by name
